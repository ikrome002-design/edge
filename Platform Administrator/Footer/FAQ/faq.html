2. User Management

How do I add users?

To add new users:

  • Go to User Management
  • Click "Add New User"
  • Enter user details and role
  • Send invitation
How do I edit users?

To edit existing users:

  • Locate the user in User Management
  • Click the edit icon
  • Modify necessary information
  • Save changes
How do I remove or change user roles?

To manage user roles:

  • Access User Management
  • Select the user
  • Click "Manage Roles"
  • Update or remove roles as needed
